BottleFall Game Engine

This Game Engine was developed as a part of the course game engine foundation (CSC591) using javascript, and html canvas.

The engine contains Sub Engines to handle Rendering, Input, Graphics, Networking, AI and Physics.

Features

  • WebGL and Canvas

    Supports 2D as well as 3D graphics rendering using HTML Canvas and WebGL.

  • Input

    Supports Input through keyboard and mouse controls.

  • Collision

    Collision detection using axis-aligned bounding box.

  • Storage

    Allows for browser storage for storing game states.

  • Physics

    Inbuilt 2D-velocity sub-engine.

  • Particle System

    Supports all kinds of particle systems through memory efficient use of game objects.

  • Networking

    Networking implemented through peerjs framework for great multiplayer experience.

  • Graphs

    Flexible support commonly used graphs.

  • AI

    Pathfinding implemented through adaptable A* and Dijkstra with support for custom heuristics.

  • Sound

    Supports multiple sounds for levels and event handlers.

  • Sprites

    Sprites supported through custom spritesheets.

  • Textures

    Supports texture mapping for spherical models, and WebGL triangles.

Getting Started

Installation

Download and include engine.js file in your html page.

<script type="text/javascript" src="engine/engine.js"></script>

For 3D, additionally include graphics.js and gl-matrix.js.

Basic Game Setup

Create Engine Object

Create a game engine object in order to access the features of the game engine.

var engine = new Engine(canvas, context, "2D");

Set Handlers

Set up the handler callbacks.


//Basic game loop handlers
engine.setUpdateHandler(gameUpdateFunction);
engine.setDrawHandler(gameDrawFunction);

//Additional event handlers
engine.input.setKeyboardPressHandler(keyPressed);

Update

Write your update function of the game loop. Make sure to call the engine's update function to update physics/collision etc. You can add some pre/post processing to the engine's update call.


function gameUpdateFunction() {
    ...
    engine.update();
    ...
}

Draw

Write your draw function of the game loop. Make sure to call the engine's draw function to draw all game objects added to the game engine.


function gameDrawFunction() {
    ...
    engine.draw();
    ...
}

Starting the game loop

Start the engine's game loop with a simple setInterval call. You can specify how frequently your game loop to be run by adjusting the frameTime parameter.

setInterval(engine.GameLoop, frameTime);
